============== SETUP APLIKASI Web Builder App ===============

1. Install Dependency dengan perintah "composer install" pada terminal (CTRL + `)
2. copy file environment "cp .env.example .env"
3. Jika key belum tersedia, generate key: "php artisan key:generate"
4. Setup database pada file ".env"
5. jalankan migration "php artisan migrate"
6. Lalu jalankan aplikasi "php artisan serve"

7. lanjut dengan menginstall project asset node "npm install"
8. jalankan perintah "npm run dev" atau "npm start"
